# Анализ настроений телеграмм-каналов популярных новостей

## Основная задача 
Разработать модель машинного обучения, которая способна автоматически определять эмоциональную окраску текстов (настрой) в сообщениях популярных новостных Телеграмм-каналов. Настроения могут быть положительными, отрицательными или нейтральными
